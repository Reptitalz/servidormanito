# **App Name**: Hey Manito! - WhatsApp Bot Manager

## Core Features:

- Marketing Page: Landing page showcasing the app's features and benefits with animated gradient waves.
- Plan Management: Display plans with credits (1 credit = 1000 messages) for the WhatsApp bot.
- Login/Registration: Implement user authentication with Google Firebase.
- Assistant Management Interface: User interface to manage WhatsApp bots and assistants.

## Style Guidelines:

- Primary color: Deep Indigo (#4F46E5) to convey trust and innovation. It's versatile and contrasts well in both light and dark schemes.
- Background color: Very light desaturated Indigo (#E5E7EB) to create a clean and modern base.
- Accent color: Violet (#8B5CF6) for interactive elements and highlights, adding a vibrant touch.
- Headline font: 'Poppins', sans-serif. Body font: 'PT Sans', sans-serif.
- Use React-icons library for a consistent and modern icon set.
- Modern and engaging layout inspired by Stripe, featuring smooth transitions and clear information hierarchy.
- Subtle animations and gradient waves to create an engaging and dynamic user experience.
